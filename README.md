# asset-reclaim

An asset recovery site, catering to financial institutions needing to recover their assets (cars, boats, rv's, homes, etc) when the end customer can no longer make their financing payments. This company, American Recovery Service, provides all asset recovery services to these financial institutions.

## Setup

cd asset-reclaim
npm i
```

npm run dev
```
